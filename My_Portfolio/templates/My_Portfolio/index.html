Text Generation using Markov Chains

App

Text Generation using Markov Chains

Markov Chain is a stochastic model in which the probability of an event occurring depends only upon preceding event.

In this application, you can generate the text by restricting the next state to top five probable states to generate more meaningful text.

The text data for this app is available on this kaggle link. The source code can be found on my GitHub account page.
